Personal Dashboard Extension

The Personal Dashboard Extension helps users manage their daily tasks, weather updates, and bookmarks directly from their browser's new tab page. This extension enhances productivity and keeps important information easily accessible.

GitHub Link

Project Overview

Key Features:
Task Management: Allows users to add, edit, and delete tasks.
Weather Updates: Displays current weather information based on the user's location.
Bookmarks: Provides quick access to user's favorite sites.
Customizable Background: Users can choose their own background images for the dashboard.
Responsive Design: Ensures the extension is accessible and user-friendly on various devices, including desktops, tablets, and smartphones.

Installation Instructions

Step 1: Clone the repository from GitHub: git clone https://github.com/NightsHigh/Personal-Dashboard.git
Step 2: Open Chrome and go to chrome://extensions/.
Step 3: Enable "Developer mode" at the top right.
Step 4: Click on "Load unpacked" and select the cloned repository folder.
Step 5: The extension should now be installed and ready to use.
