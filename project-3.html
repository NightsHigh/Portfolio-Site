Quizzical

Quizzical is an interactive quiz application that allows users to test their knowledge on various topics. The app provides a fun and engaging way for users to learn new information and challenge themselves with multiple-choice questions.

Live Link

Project Overview

Key Features:
Topic Selection: Users can choose from a variety of quiz topics.
Question Display: Presents multiple-choice questions one at a time, with immediate feedback on correct or incorrect answers.
Score Tracking: Keeps track of the user's score and displays the results at the end of the quiz.
Timer: Adds a timer for each question to increase the challenge.
Responsive Design: Ensures the application works well on different devices, including desktops, tablets, and smartphones.
